ROI analyzer

Build and install ROI analyzer to your BoneJ (or FIJI) installation
Adds a number of plug-ins under IVDisk, try the ROI Export.
Saves the current ROI in a text file to c:\Output\ROIs\StackName\SliceName_meanY_meanX.txt as tab-separated text file, creates the directory structure if it doesn't exist (be aware...). Will overwrite any file with identical name.
